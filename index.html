OpenCore Installation Guide

Complete guide for installing macOS on compatible hardware

OpenCore is a modern and powerful bootloader used to run macOS on non-Apple computers, commonly known as Hackintosh systems. It focuses on stability, security, and accuracy, making the system behave as close as possible to a real Mac.

Before starting with OpenCore, it's important to understand your computer's hardware. Things like the CPU, GPU, motherboard, and Wi-Fi card must be compatible with macOS. Without proper hardware support, macOS may not boot or work correctly.

The next step is preparing the required files. OpenCore uses an EFI folder that contains essential components such as drivers, ACPI files, and the main configuration file called config.plist. This file controls how macOS communicates with your hardware, so careful setup is very important.

Once the EFI folder is ready, it is placed on a USB installer or directly on the system's EFI partition. After booting through OpenCore, macOS installation can begin just like on a real Mac.

Although OpenCore can seem difficult at first, following official documentation and guides step by step makes the process much easier. With patience and attention to detail, OpenCore provides a reliable way to experience macOS on compatible hardware.

Installation Steps

Follow these steps carefully to configure your OpenCore installation.

1
Extract the OpCore-Simplify package and launch the application inside. Follow the on-screen instructions to create your EFI folder structure.
2
Run the USB Toolbox tool to generate the required kext file for your system's USB configuration.
3
Download the USB Toolbox kext and place both kext files into the EFI > OC > Kext directory.
4
Open the OCAT application from its folder, then press Ctrl+O to open your config.plist file.
5
Click Kext in the sidebar, then click the "+" button to add the two new kext files you just placed.
6
Proceed to the next section to create your macOS installer. You're almost ready! üéâ

Putting Everything Together

You're almost ready to install macOS! Follow these final steps to complete your bootable USB and begin installation.

1
Format your USB flash drive as FAT32. You can use Rufus or Windows Disk Management for this. Make sure to backup any data on the USB first as formatting will erase everything.
2
Copy both the 'com.apple.recovery.boot' folder (created by macrecovery) and your 'EFI' folder to the root of the USB flash drive.
3
Restart your computer and enter the BIOS/UEFI settings (usually by pressing F2, F10, F12, or Delete during startup).
4
In BIOS settings, disable Secure Boot and disable Fast Boot. These features can prevent OpenCore from loading properly.
5
Save your BIOS changes and select your USB flash drive as the boot device. Your system should now boot into the OpenCore menu.
6
Select the macOS installer from the OpenCore menu and proceed with the macOS installation. Follow the on-screen instructions just like installing on a real Mac. When complete, you're done! üéâ
